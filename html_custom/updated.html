Replace USD with THB for AliExpress.com
By PoundXI (poundxi.com)

ส่วนขยายได้รับการอัปเดต v

ส่วนขยายนี้ยุติการพัฒนาแล้ว !! เวอร์ชั่นนี้จะเป็นเวอร์ชั่นสุดท้ายของการอัปเดต

เนื่องจากช่วงเดือนมกราคม ปี 2019 เว็บไซต์ AliExpress.com สามารถแสดงค่าเงินบาทในตัวเองได้แล้ว

และการแสดงผลค่าเงินบาทของ AliExpress.com ทำได้ดีขึ้นอย่างต่อเนื่อง ผมจึงไม่มีความจำเป็นที่ต้องพัฒนาส่วนขยายนี้อีกต่อไป

หลังจากยุติการพัฒนาแล้ว ผมก็ได้นำ source code ของส่วนขยายนี้ไปเผยแพร่ไว้บน GitHub เพื่อให้ผู้ที่สนใจสามารถนำไปพัฒนาต่อได้ตามเงื่อนไข

ผมจึงขอประกาศยุติการพัฒนาไว้แต่เพียงเท่านี้, ขอบคุณทุกท่านที่สนับสนุนการใช้งานตลอดมา

หมายเหตุ

  • คุณยังสามารถใช้ส่วนขยายต่อไปได้ ตราบใดที่ส่วนขยายยังแสดงค่าเงินบาทให้คุณ
  • วิธีแสดงเงินบาทแบบไม่ใช้ส่วนขยาย
    • เปิดเว็บไซต์ https://th.aliexpress.com
    • คลิกที่เมนู จัดส่งไปยัง ที่มุมขวาด้านบน
    • เลือกจัดส่งไปยัง Thailand
    • เลือกสกุลเงิน THB ( Thai Baht )
    • คลิกปุ่ม บันทึก
  • คุณสามารถลบส่วนขยายออกได้ทุกเมื่อที่คุณต้องการ
